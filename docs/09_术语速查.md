# 📖 术语速查

> 计算广告核心术语速查表

---

## 📊 计费与结算术语

| 术语 | 英文全称 | 中文含义 | 说明 |
|-----|---------|---------|------|
| **CPT** | Cost Per Time | 按时间计费 | 按展示时长收费 |
| **CPM** | Cost Per Mille | 千次展示成本 | 按1000次展示收费 |
| **CPC** | Cost Per Click | 单次点击成本 | 按点击次数收费 |
| **CPA** | Cost Per Action | 单次行动成本 | 按转化行为收费 |
| **CPS** | Cost Per Sale | 单次销售成本 | 按成交收费 |
| **CPI** | Cost Per Install | 单次安装成本 | 按App安装收费 |
| **CPL** | Cost Per Lead | 单次线索成本 | 按销售线索收费 |
| **CPV** | Cost Per View | 单次观看成本 | 按视频观看收费 |
| **eCPM** | effective CPM | 有效千次展示收入 | 实际每千次展示收入 |

## 📈 效果指标术语

| 术语 | 英文全称 | 中文含义 | 计算公式 |
|-----|---------|---------|---------|
| **CTR** | Click-Through Rate | 点击率 | 点击数/展示数 |
| **CVR** | Conversion Rate | 转化率 | 转化数/点击数 |
| **ROI** | Return On Investment | 投资回报率 | (收入-成本)/成本 |
| **ROAS** | Return On Ad Spend | 广告支出回报率 | 收入/广告花费 |
| **LTV** | Lifetime Value | 生命周期价值 | 用户全生命周期收入 |
| **ARPU** | Average Revenue Per User | 用户平均收入 | 总收入/用户数 |
| **DAU** | Daily Active Users | 日活跃用户 | 每日活跃用户数 |
| **MAU** | Monthly Active Users | 月活跃用户 | 每月活跃用户数 |

## 🏢 市场参与者术语

| 术语 | 英文全称 | 中文含义 | 说明 |
|-----|---------|---------|------|
| **Advertiser** | - | 广告主 | 投放广告的企业/个人 |
| **Publisher** | - | 媒体/发布商 | 提供广告位的网站/App |
| **Agency** | - | 代理公司 | 代理广告投放的公司 |

| **SSP** | Supply-Side Platform | 供给方平台 | 帮助媒体销售流量 |
| **ADX** | Ad Exchange | 广告交易平台 | 连接DSP和SSP |
| **DMP** | Data Management Platform | 数据管理平台 | 用户数据管理 |
| **TD** | Trading Desk | 交易终端 | 广告采买平台 |
| **ADN** | Ad Network | 广告网络 | 聚合媒体流量 |

## ⚠️ 反作弊术语

|-----|---------|---------|------|

| **GIVT** | General Invalid Traffic | 一般无效流量 | 可被规则识别 |
| **SIVT** | Sophisticated Invalid Traffic | 复杂无效流量 | 需要算法识别 |
| **Bot** | - | 机器人流量 | 程序生成的假流量 |
| **Click Fraud** | - | 点击欺诈 | 恶意点击行为 |
| **Ad Fraud** | - | 广告欺诈 | 广告作弊行为 |
| **Brand Safety** | - | 品牌安全 | 避免广告出现在不当内容旁 |

## 🏷️ 常用缩写速查

```
广告类型：
• DA = Display Advertising (展示广告)
• SA = Search Advertising (搜索广告)
• VA = Video Advertising (视频广告)
• NA = Native Advertising (原生广告)
• PA = Programmatic Advertising (程序化广告)

平台类型：
• DSP = Demand-Side Platform
• SSP = Supply-Side Platform
• ADX = Ad Exchange
• DMP = Data Management Platform
• CDP = Customer Data Platform

指标类型：
• KPI = Key Performance Indicator
• GMV = Gross Merchandise Value
• AOV = Average Order Value
• CAC = Customer Acquisition Cost

技术相关：
• ML = Machine Learning
• AI = Artificial Intelligence
• NLP = Natural Language Processing
• CV = Computer Vision
```
