# 📖 搜索与竞价广告

> 效果广告的核心交易机制

---

## 🎯 什么是竞价广告？

**竞价广告**是指通过**实时竞价**的方式决定广告投放的交易模式。广告主对目标人群出价，系统根据出价和质量得分决定广告展示。

### **与合约广告的本质区别**

| 特点 | 合约广告 | 竞价广告 |
|-----|---------|---------|
| **定价方式** | 预先谈判确定 | 市场竞争决定 |
| **量的保证** | 媒体承诺完成 | 无保证，竞价获取 |
| **适用对象** | 大品牌广告主 | 中小广告主为主 |
| **决策逻辑** | 计划调度 | 实时竞价 |

## 💰 竞价机制：GSP

### **GSP（Generalized Second-Price）广义第二价格**

**核心规则**：按照下一位广告主的出价来计费，而不是自己的出价。

```
示例：关键词"手机"的竞价结果

┌────────────┬────────────┬────────────┬────────────┬────────────┐
│   广告主    │    出价     │    质量分   │   排序分    │   实际CPC   │
│            │   (bid)    │   (QS)     │  (bid×QS)  │            │
├────────────┼────────────┼────────────┼────────────┼────────────┤
│   华为      │   ¥5.0     │    1.0     │    5.0     │   ¥3.2     │
│   小米      │   ¥4.0     │    0.8     │    3.2     │   ¥2.5     │
│   OPPO     │   ¥3.0     │    0.8     │    2.4     │   ¥1.6     │
│   vivo     │   ¥2.0     │    0.8     │    1.6     │   最低价    │
└────────────┴────────────┴────────────┴────────────┴────────────┘

华为的实际CPC = (小米排序分 / 华为质量分) + 0.01
              = (3.2 / 1.0) + 0.01 = ¥3.21
```

### **为什么用第二价格？**

```
第二价格竞价的优势：

1. 鼓励真实出价
   广告主没有动力低报价（可能错失展示）
   广告主没有动力高报价（不会按高价计费）
   
2. 市场效率更高
   真实反映市场供需
   
3. 减少博弈成本
   广告主无需频繁调价来"试探"市场
```

## 📊 广告排序：eCPM

### **排序公式**

$$排序分 = bid \times CTR \times κ$$

其中：
- **bid**：广告主出价
- **CTR**：预估点击率
- **κ**：调节因子（考虑其他因素）

> 本质上就是 eCPM 排序

### **质量得分（Quality Score）**

质量得分是搜索广告系统的核心概念，反映广告质量。

```
质量得分的组成因素：

┌─────────────────────────────────────────────────────────┐
│                    质量得分 (1-10)                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   ┌─────────────────┐                                   │
│   │    预期点击率     │  历史CTR表现                       │
│   │     (40%)       │                                   │
│   └─────────────────┘                                   │
│                                                         │
│   ┌─────────────────┐                                   │
│   │    广告相关性     │  广告与搜索词的匹配程度             │
│   │     (30%)       │                                   │
│   └─────────────────┘                                   │
│                                                         │
│   ┌─────────────────┐                                   │
│   │   落地页体验      │  落地页与广告的相关性和质量         │
│   │     (30%)       │                                   │
│   └─────────────────┘                                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### **质量得分的影响**

| 质量得分 | 对CPC的影响 | 对排名的影响 |
|---------|-----------|-------------|
| 高（8-10） | CPC更低 | 更容易获得靠前位置 |
| 中（5-7） | 正常CPC | 正常排名 |
| 低（1-4） | CPC更高 | 可能无法展示 |

## 🌐 广告网络（Ad Network）

### **什么是广告网络？**

广告网络是**聚合多个媒体的剩余流量**，按照人群标签进行售卖的产品。

```
┌─────────────────────────────────────────────────────────┐
│                     广告网络架构                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   供给方（媒体）                     需求方（广告主）        │
│                                                         │
│   ┌─────────┐                      ┌─────────┐         │
│   │ 网站A    │                      │ 广告主1  │         │
│   └────┬────┘                      └────┬────┘         │
│        │                                │              │
│   ┌────┴────┐    ┌─────────────┐   ┌────┴────┐        │
│   │ 网站B    │───▶│   广告网络   │◀──│ 广告主2  │        │
│   └────┬────┘    │             │   └────┬────┘        │
│        │         │  • 流量聚合  │        │              │
│   ┌────┴────┐    │  • 人群定向  │   ┌────┴────┐        │
│   │ 网站C    │    │  • 竞价分配  │   │ 广告主3  │        │
│   └─────────┘    └─────────────┘   └─────────┘        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### **广告网络的特点**

| 特点 | 说明 |
|-----|------|
| **淡化媒体** | 广告主购买的是人群，不是具体位置 |
| **竞价模式** | 按eCPM排序，无量的保证 |
| **CPC结算** | 主流结算方式 |
| **长尾流量** | 主要消耗媒体的剩余流量 |

### **产品案例：Google Display Network**

```
GDN（Google Display Network）：
- 覆盖全球200万+网站
- 可触达90%的互联网用户
- 定向方式：
  ├── 关键词定向
  ├── 主题定向
  ├── 兴趣定向
  ├── 再营销定向
  └── 相似受众定向
```

## 📈 效果评估

### **核心指标**

```
广告漏斗指标：

曝光 (Impressions)
    │
    │ CTR (点击率)
    ▼
点击 (Clicks)
    │
    │ CVR (转化率)
    ▼
转化 (Conversions)
    │
    │ 客单价
    ▼
收入 (Revenue)

关键公式：
• CPC = 花费 / 点击数
• CTR = 点击数 / 曝光数
• CVR = 转化数 / 点击数
• CPA = 花费 / 转化数
• ROAS = 收入 / 花费
```

### **ROI 计算**

```
示例：
─────────────────────────────────────
广告花费：¥10,000
点击数：2,000
转化数：100
客单价：¥300
─────────────────────────────────────
CPC = 10,000 / 2,000 = ¥5
CTR = 假设曝光10万，则CTR = 2%
CVR = 100 / 2,000 = 5%
CPA = 10,000 / 100 = ¥100
收入 = 100 × 300 = ¥30,000
ROAS = 30,000 / 10,000 = 3.0 (300%)
ROI = (30,000 - 10,000) / 10,000 = 200%
─────────────────────────────────────
```

## ⚖️ 合约广告与竞价广告的比较

```
┌────────────────────────────────────────────────────────────┐
│                 合约 vs 竞价 对比                            │
├─────────────────────────────┬──────────────────────────────┤
│         合约广告             │          竞价广告             │
├─────────────────────────────┼──────────────────────────────┤
│ 类似"计划经济"               │ 类似"市场经济"                │

├─────────────────────────────┼──────────────────────────────┤
│ 适合：品牌广告               │ 适合：效果广告                │
│      大型广告主              │      中小广告主               │
│      需要量保证              │      追求ROI                 │
└─────────────────────────────┴──────────────────────────────┘
```
