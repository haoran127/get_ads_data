# 📖 计算广告基础

> 理解在线广告的本质和核心问题

---

## 🎯 什么是计算广告？

**计算广告**（Computational Advertising）是一门利用大规模计算技术解决广告投放问题的学科。

### **核心定义**
> 计算广告的核心问题是：**为一系列用户与环境的组合找到最合适的广告投放策略以优化整体利润**

用数学语言表达：

$$\max \sum_{i} r(a_i, u_i, c_i) - cost$$

其中：
- **a (Ad)**: 广告
- **u (User)**: 用户
- **c (Context)**: 上下文环境
- **r**: 收益函数
- **cost**: 广告系统运营成本

## 📊 广告的有效性

### **广告效果的决定因素**

广告效果主要取决于以下因素的组合：

```
广告效果 = f(广告创意, 受众匹配度, 投放环境, 投放时机)
```

### **广告预算的分解**

广告预算可以分为三部分：

```
┌─────────────────────────────────────────────────────┐
│                 广告预算总额                          │
├───────────────┬─────────────────┬───────────────────┤
│   有效预算     │    浪费预算      │    负效预算        │
│   (Effective) │   (Wasted)      │   (Negative)      │
├───────────────┼─────────────────┼───────────────────┤
│ 带来正向效果   │ 没有任何效果     │ 产生负面影响       │
│ 触达目标用户   │ 触达非目标用户   │ 品牌安全问题       │
│ 产生转化      │ 无法衡量        │ 过度曝光           │
└───────────────┴─────────────────┴───────────────────┘

计算广告的目标：最大化有效预算占比，减少浪费和负效预算
```

## 🔑 关键概念：eCPM

### **eCPM 是什么？**

**eCPM**（effective Cost Per Mille）= 期望千次展示收入

这是广告系统内部用于比较不同广告价值的统一标准。

### **eCPM 计算公式**

| 结算方式 | eCPM 计算公式 |
|---------|-------------|
| **CPM** | eCPM = bid |
| **CPC** | eCPM = CTR × bid × 1000 |
| **CPA** | eCPM = CTR × CVR × bid × 1000 |

> **bid**: 出价
> **CTR**: 点击率 (Click-Through Rate)
> **CVR**: 转化率 (Conversion Rate)

### **eCPM 的重要性**

```
为什么需要 eCPM？

广告A：按CPM出价，¥10/千次展示
广告B：按CPC出价，¥0.5/点击，预估CTR = 1%

如何比较？转换为统一标准：

广告A的eCPM = ¥10
广告B的eCPM = 1% × ¥0.5 × 1000 = ¥5

结论：应该优先展示广告A
```

## 📈 广告收入公式

对于媒体/广告平台而言：

$$收入 = 流量 \times eCPM$$

展开来看：

$$收入 = 展示次数 \times CTR \times 点击价值$$

### **提升收入的三个杠杆**

| 杠杆 | 提升方法 | 负责部门 |
|-----|---------|---------|
| **流量** | 提升DAU、用户时长、广告位数量 | 产品运营 |
| **CTR** | 优化创意、提升相关性、改善定向 | 广告技术 |
| **点击价值** | 吸引高质量广告主、优化竞价 | 商业化 |

## 🎯 广告效果的衡量

### **效果类广告指标**

```
曝光 → 点击 → 访问 → 转化 → 复购
 │      │      │      │      │
 ▼      ▼      ▼      ▼      ▼
CPM    CPC    CPV    CPA   CLV
```

| 指标 | 全称 | 含义 |
|-----|-----|------|
| **CPM** | Cost Per Mille | 千次曝光成本 |
| **CPC** | Cost Per Click | 单次点击成本 |
| **CPA** | Cost Per Action | 单次转化成本 |
| **CTR** | Click-Through Rate | 点击率 = 点击/曝光 |
| **CVR** | Conversion Rate | 转化率 = 转化/点击 |
| **ROI** | Return On Investment | 投资回报率 |
| **ROAS** | Return On Ad Spend | 广告支出回报率 |

### **品牌类广告指标**

| 指标 | 含义 | 衡量方法 |
|-----|------|---------|
| **Reach** | 触达人数 | 去重后的曝光用户数 |
| **Frequency** | 频次 | 平均每人看到广告的次数 |
| **GRP** | 毛评点 | Reach × Frequency |
| **Brand Lift** | 品牌提升 | 调研对比测试组和对照组 |

## 🔄 在线广告的交易方式

### **交易方式演进**

```
   时间线
     │
     │    ┌─────────────────┐
     │    │  按时间段包断     │  CPT
     │    │  (早期门户)      │
     │    └────────┬────────┘
     │             │
     │    ┌────────▼────────┐
     │    │  展示量合约      │  CPM
     │    │  (品牌广告)      │
     │    └────────┬────────┘
     │             │
     │    ┌────────▼────────┐
     │    │  竞价广告        │  CPC/CPA
     │    │  (搜索/效果)     │
     │    └────────┬────────┘
     │             │
     │    ┌────────▼────────┐
     ▼    │  程序化交易      │  RTB
          │  (实时竞价)      │
          └─────────────────┘
```

### **主要交易方式对比**

| 交易方式 | 特点 | 适用场景 | 结算方式 |
|---------|-----|---------|---------|
| **合约广告** | 预先约定量和价格 | 品牌广告 | CPT/CPM |
| **竞价广告** | 实时竞价决定 | 效果广告 | CPC/CPA |
| **RTB** | 逐次展示实时竞价 | 精准定向 | CPM |
| **PMP** | 私有市场优先竞价 | 优质流量 | CPM |
