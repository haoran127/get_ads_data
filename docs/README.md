## 🔑 核心公式

### **计算广告的核心优化目标**

$$\max \sum_{i} r(a_i, u_i, c_i) - cost$$

**关键变量：**
- **a (Ad)**: 广告
- **u (User)**: 用户  
- **c (Context)**: 上下文/环境
- **r**: 收益函数
- **cost**: 成本

### **eCPM 计算公式**

$$eCPM = CTR \times bid \times 1000$$

- **eCPM**: 期望千次展示收入
- **CTR**: 点击率
- **bid**: 出价
