# 广告与素材优化系统快速指南

## 一、核心优化策略

### 1.1 广告优化策略矩阵

| 广告类型 | ROAS范围 | 策略 | 执行动作 |
|---------|---------|------|---------|
| **高绩效** | > 目标值 × 1.3 | 扩大投放 | 增加出价10-20%<br/>增加预算<br/>复制到相似广告组 |
| **中绩效** | 目标值 ± 20% | 微调优化 | 小幅调整出价<br/>A/B测试<br/>优化素材组合 |
| **低绩效** | < 目标值 × 0.7 | 快速淘汰 | 暂停广告<br/>记录原因 |
| **新广告** | 数据不足 | 加速测试 | 临时提高出价<br/>7天后评估 |

### 1.2 素材优化策略矩阵

| 素材类型 | ROAS范围 | 策略 | 执行动作 |
|---------|---------|------|---------|
| **高绩效** | > 平均 × 1.3 | 推广使用 | 应用到更多广告组<br/>优先在相似组使用 |
| **低绩效** | < 平均 × 0.7 | 替换移除 | 从低绩效广告移除<br/>用高绩效素材替换 |
| **最优组合** | 组合ROAS高 | 组合优化 | 创建新广告使用组合<br/>测试不同组合 |

## 二、优化工作流

### 2.1 日常优化循环

```
数据采集 (每15分钟-1小时)
    ↓
性能分析 (广告 + 素材)
    ↓
生成优化建议
    ↓
风险评估
    ↓
执行优化 (批量操作)
    ↓
监控效果 (实时)
    ↓
反馈循环
```

### 2.2 规模化扩展流程

```
识别高ROI机会
    ↓
分析成功因素
    ↓
小规模测试 (10%预算, 3-7天)
    ↓
验证效果
    ↓
扩大规模 (50%预算, 3-7天)
    ↓
再次验证
    ↓
全量扩展 (100%预算)
    ↓
持续监控
```

## 四、优化决策树

### 4.1 广告优化决策

```
广告性能分析
    ├─ ROAS > 目标值 × 1.3
    │   ├─ 趋势上升/稳定 → 扩大投放
    │   └─ 趋势下降 → 密切监控
    │
    ├─ ROAS 在目标值 ± 20%
    │   └─ 微调优化 (出价调整/A/B测试)
    │
    ├─ ROAS < 目标值 × 0.7
    │   ├─ 数据充足 → 暂停广告
    │   └─ 数据不足 → 等待更多数据
    │
    └─ 新广告 (< 7天)
        └─ 加速测试 (临时提高出价)
```

### 4.2 素材优化决策

```
素材性能分析
    ├─ 高绩效素材
    │   ├─ 使用率低 → 推广到更多广告组
    │   └─ 使用率高 → 监控保持
    │
    ├─ 低绩效素材
    │   ├─ 使用率高 → 替换素材
    │   └─ 使用率低 → 移除素材
    │
    └─ 素材组合分析
        ├─ 找到最优组合 → 创建新广告
        └─ 需要新创意 → 生成新素材
```

## 五、预算分配策略

### 5.1 基于ROAS的预算分配

```
ROAS排序
    ↓
高ROAS (> 目标值 × 1.2)
    → 增加预算 (最高到上限)
    ↓
中ROAS (目标值 ± 20%)
    → 保持或小幅增加预算
    ↓
低ROAS (< 目标值 × 0.8)
    → 减少预算 (最低到下限)
    ↓
验证整体ROAS
    ↓
如果未满足目标 → 调整分配比例
```

### 5.2 动态预算调整规则

- **高ROAS组预算即将耗尽** → 自动增加预算
- **低ROAS组预算剩余** → 转移到高ROAS组
- **预算使用率 > 90%** → 发送告警

## 六、风险控制检查点

### 6.1 变更前检查
- ✅ ROAS预测 > 目标值
- ✅ 预算充足
- ✅ 账户状态正常

### 6.2 变更后监控
- ✅ 性能下降 > 10% → 自动回滚
- ✅ 性能正常 → 继续监控
- ✅ 性能提升 → 记录成功

## 七、关键指标监控

### 7.1 实时监控指标（每15分钟）

### 7.3 深度分析指标（每周）

## 八、自动化规则示例

### 规则1：低绩效广告自动暂停
```
条件：
- ROAS < 目标值 × 0.7
- 转化数 = 0 且 点击 >= 20
- 投放时间 >= 7天

动作：
- 暂停广告
- 记录原因
```

### 规则2：高绩效广告扩大投放
```
条件：
- ROAS > 目标值 × 1.3
- 转化数 >= 5
- 趋势：上升或稳定

动作：
- 增加出价 10-20%
- 增加预算
- 复制到相似广告组
```

### 规则3：高绩效素材推广
```
条件：
- 素材ROAS > 平均ROAS × 1.3
- 使用该素材的广告数量 < 5

动作：
- 应用到更多广告组
- 优先在相似广告组使用
```

### 规则4：预算自动分配
```
条件：
- 高ROAS组预算使用率 > 90%
- 低ROAS组预算剩余 > 20%

动作：
- 从低ROAS组转移预算到高ROAS组
- 保持整体ROAS目标
```
